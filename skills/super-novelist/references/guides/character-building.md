# 人物塑造指南

好的人物是故事的灵魂。读者记住的是人，不是情节。

> 人物档案模板请使用 [character-template.md](character-template.md)（Phase 2 创建时使用）。本文档聚焦于**在写作过程中**塑造人物的技法。

---

## 核心原则：矛盾创造深度

真实的人是复杂的，充满矛盾。矛盾不是缺陷，而是人物的**发动机**。

**有效矛盾示例：**
- 雷厉风行的女警长，回家后对着亡夫的照片说话
- 满口仁义的书生，暗中替人写休书赚银子
- 冷血杀手，每天给流浪猫带鱼
- 最不信命的人，抽屉里藏着一张求来的护身符

**无效矛盾**：为矛盾而矛盾，与人物核心无关。

## 侧面揭示技法

不要直接陈述性格，通过行为展示。这是"展示而非讲述"在人物塑造上的核心应用。

### 用动作替代标签

| 直接陈述（弱） | 侧面展示（强） |
|----------------|----------------|
| 他很愤怒 | 他捏碎手中的纸杯，碎片扎进掌心也没松手 |
| 她很紧张 | 她反复调整眼镜位置，第三次了 |
| 他很傲慢 | 他从不直视下属的眼睛，对话时总是看窗外 |
| 她很善良 | 她偷偷喂了三年流浪狗，谁也没告诉 |
| 他很孤独 | 电视永远开着，他从来不看 |

### 用选择揭示本质

人物在压力下的选择，比任何描述都有说服力。

**示例：**
> 火灾现场，他可以救名画，也可以救那只猫。他抱起了猫。
> 不需要写"他善良"，这个选择已经说了。

### 用习惯暴露真相

反复出现的小习惯是人物的无意识自我暴露。

- 紧张时摸无名指（曾经戴过戒指？）
- 总是坐在背靠墙的位置（缺乏安全感？）
- 说话时避免眼神接触（隐瞒什么？）
- 对每个服务员都说谢谢（教养？还是内疚？）

## 主角塑造

### 主角必须有

- **明确目标** - 他想要什么
- **强大动机** - 为什么想要
- **可共情性** - 读者能理解他的感受
- **成长空间** - 故事中会改变

### 主角原型

| 类型 | 特征 | 故事作用 |
|-----|------|---------|
| 英雄型 | 勇敢、正义、利他 | 战胜外在威胁 |
| 成长型 | 从弱小到强大 | 克服内在缺陷 |
| 反英雄型 | 道德灰色、复杂 | 挑战传统道德 |
| 平凡型 | 普通人卷入非凡事 | 读者代入感强 |

### 缺陷致命化

主角必须有缺陷，缺陷在关键时刻导致失败。没有缺陷的角色不会成长。

| 缺陷 | 导致的失败 | 成长方向 |
|-----|----------|---------|
| 傲慢 | 低估对手，落入陷阱 | 学会倾听 |
| 信任问题 | 拒绝帮助，孤立无援 | 学会信任 |
| 完美主义 | 无法及时行动，错失机会 | 学会接受不完美 |
| 复仇心 | 被利用，失去理智 | 学会放下 |
| 逃避 | 问题越滚越大 | 学会面对 |

## 反派塑造

好反派不是坏人，是**信念不同的人**。

### 好反派的标准

- **强大可信** - 不应该是草台班子
- **有自己的逻辑** - 他相信自己在做正确的事
- **与主角有深层联系** - 不是单纯为了作恶
- **揭示主题** - 挑战主角的信念

### 反派动机层次

**浅层动机**（不够）：
> "我想毁灭世界" → 乏味

**中层动机**（及格）：
> "我失去了一切，世界对我没有意义" → 可理解但单薄

**深层动机**（有力）：
> "我的女儿死于一种本可治愈的病，但药厂为了利润延迟了上市。这个世界不值得被拯救。" → 读者虽然不认同，但能理解

### 反派与主角的镜像关系

最好的反派是主角的**黑暗镜像**——如果主角走了另一条路，就会变成他。

```
主角：失去家人后选择保护他人
反派：失去家人后选择毁灭一切
同一起点，不同选择。
```

## 配角的功能性

配角存在的唯一理由是**对主角产生作用**。

### 配角类型

| 类型 | 功能 | 注意 |
|-----|------|------|
| 导师型 | 指引主角，传递信息 | 必须有自己的盲区 |
| 盟友型 | 协助主角，提供支持 | 不能只是工具人 |
| 搞笑型 | 缓解紧张 | 在关键时刻也能严肃 |
| 叛徒型 | 制造转折 | 背叛要有前期铺垫 |

### 配角原则

- 删除"只是存在"的角色
- 每个配角必须有明确作用
- 避免刻板印象（除非是有意为之）

## 人物关系设计

### 关系动态变化

好的关系会随故事发展，不是静态的：

```
第一章：陌生人
第三章：不情愿的盟友
第五章：建立信任
第七章：背叛/考验
终章：真正的友谊（或决裂）
```

### 关系揭示层次

逐步揭示关系深度，不要一次性全部暴露：
- **表层**：表面互动
- **中层**：共同经历后的默契
- **深层**：真实感受/秘密

### 高戏剧潜力关系

| 关系类型 | 戏剧潜力 | 经典模式 |
|---------|---------|---------|
| 亦敌亦友 | 高 | "我知道你在利用我，但我需要你" |
| 禁忌之爱 | 高 | 身份/立场对立，但无法控制感情 |
| 师徒关系 | 中 | "教会徒弟，饿死师父"的潜在张力 |
| 兄弟竞争 | 中 | 同一起点，不同选择，此消彼长 |

## 人物出场设计

### 首次出场原则

有效的出场方式：
- **行动中** - 展示能力或性格
- **冲突中** - 立即建立关系/对立
- **误解中** - 建立悬念

### 出场示例对比

| 无效出场 | 有效出场 |
|---------|---------|
| 李明，28岁，是一名侦探。他走进办公室。| 李明跨过警戒线，警官试图拦住他。"市刑警队，李明。"他亮出证件，径直走向尸体。|
| 美丽的女孩坐在窗边，她叫小红。| 她已经三天没睡了，咖啡杯里的液体在颤抖。当门铃响起时，她几乎把杯子摔在地上。|

**避免**：镜子自照描写外貌、简历式介绍、无意义的日常活动。

## 人物一致性检查

角色行为必须符合已建立的性格。

**检查问题：**
- 这件事符合他的核心价值观吗？
- 以他的背景，会有这样的反应吗？
- 他的恐惧会如何影响这个决定？
- 他的缺陷会导致他犯什么错？

**例外处理：**
- 如果角色"不符合性格"行事，必须有原因
- 解释应该在相同/下一章提供
- 可以是成长的标志（角色克服了缺陷）
