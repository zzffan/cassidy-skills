# 对话写作规范

好对话是揭示人物、推动情节、制造冲突的有力工具。

---

## 对话核心原则

### 1. 对话必须有目的

每句对话应该至少完成以下之一：

| 目的 | 示例 |
|-----|------|
| **推动情节** | "我找到凶器了，在河边的草丛里。" |
| **揭示人物** | "我不信任警察，他们从来不帮我这样的人。" |
| **制造冲突** | "你骗了我。你从头到尾都在骗我。" |
| **传达信息** | "炸弹将在三点引爆。" |
| **表达情感** | "我...我不知道该说什么。" |
| **制造悬念** | "你知道那天晚上真正发生了什么吗？" |

**无效对话：**
> "你好。"
> "你好。"
> "吃了吗？"
> "吃了。"

### 2. 对话应该简洁

人们说话不写论文。

| 啰嗦 | 简洁 |
|-----|------|
| "我想告诉你的是，我认为我们应该立刻离开这里。" | "我们得马上走。" |
| "我非常抱歉，但我真的不知道你刚才说的那件事的答案。" | "我不知道。" |

### 3. 真实的人不会完整表达

真实对话充满打断、迟疑、话题转移、话没说完、暗示而非明说。

**示例：**
> "我本来想告诉你，但是——"
> "但是什么？"
> "算了，没什么。"
> "不，你说。"
> "真的没什么。"

## 对话格式规范

### 中文对话标点

```
"说话内容，" 他说。
"说话内容？" 她问。
"说话内容！" 他大喊。
```

### 对话标签使用

- 能辨识说话人时，省略标签
- 使用"说""问"等中性标签
- 避免过度使用副词修饰

| 过度使用 | 改进后 |
|---------|--------|
| "你骗了我，"他愤怒地说。| "你骗了我。"他的声音在颤抖。 |
| "好的，"她高兴地同意道。| 她眼睛亮了。"好的。" |

### 段落划分

每个说话人的对话开始新段落。

## 潜台词（Subtext）

好的对话，真正含义在表面之下。

### 直接 vs 潜台词

| 直接（乏味） | 潜台词（有趣） |
|-------------|---------------|
| "我很生气。" | "没事。我挺好的。真的。" |
| "我喜欢你。" | "你今天看起来...不错。" |
| "我不信任你。" | "谢谢你告诉我。我会记住的。" |

### 潜台词技巧

**1. 话题转移**
```
"你爱我吗？"
"你看了天气预报吗？明天有雨。"
```

**2. 反问而非回答**
```
"你杀了他吗？"
"你觉得像我这样的人会做那种事？"
```

**3. 谈论其他事物**
```
"你想我吗？"
"我妈昨天打电话来了。"
```

**4. 沉默和动作**
```
"你愿意原谅我吗？"
她继续看杂志，翻了一页。
```

## 对话与动作结合

动作与对话一致或矛盾，都有戏剧效果。

**一致（增强）：**
> "我爱你。"她紧紧抱住他，眼泪流下来。

**矛盾（揭示真相）：**
> "我完全支持你。"他目光看向别处，手在口袋里握紧拳头。

**动作替代标签：**
```
"你在撒谎。"她拍案而起。
"坐下。"他头也不抬。
```

## 对话节奏进阶

对话不仅是内容，节奏本身就是表达。

### 快节奏对话

**特征**：短句、无标签、打断、快速升级
**适用**：争吵、追逐、紧急决策

```
"左边！"
"来不及了——"
"跳！"
水面越来越近。
"抓紧！"
```

**制造方式**：
- 去掉所有对话标签（读者能分辨说话人）
- 单句成段，不超过15字
- 允许不完整的句子
- 用动作替代呼吸空间

### 慢节奏对话

**特征**：长句、停顿、迂回、潜台词丰富
**适用**：告白、试探、审讯、告别

```
"这些年，"他端起茶杯，看着水面上的茶叶慢慢沉下去，"你有没有想过，如果当初我们没走那条路——"

"没有。"她很快回答。太快了。

他放下茶杯。"我也是。"
沉默蔓延开来。窗外的梧桐叶被风卷起，又落下。
"其实，"她说，声音低了下去，"有时候会。"
```

**制造方式**：
- 在对话间插入动作描写和沉默
- 让角色绕弯子，欲言又止
- 用环境细节控制呼吸感
- 关键信息延迟释放

### 沉默的戏剧力量

对话中最有力的不是说了什么，而是**没说什么**。

```
"你是不是早就知道了？"
他不说话。拿起杯子喝水，放下，又拿起。
"我在问你话。"
他终于看向她。"你真想听答案？"
```

沉默的用法：
- **拒绝回答** → 暗示隐瞒或保护
- **反应延迟** → 暗示震惊或权衡
- **对话中断** → 暗示情感崩溃或决心
- **以动作代替回答** → 比语言更有力

### 对话中的权力博弈

对话是权力的角力场。谁提问、谁回避、谁打断、谁沉默——每个选择都暗示地位。

| 权力信号 | 高位角色 | 低位角色 |
|---------|---------|---------|
| 提问 vs 回答 | 提问（控制话题） | 被迫回答 |
| 打断 vs 被打断 | 随意打断 | 小心不被打断 |
| 话题控制 | 决定谈什么 | 被动跟随 |
| 沉默使用 | 用沉默施压 | 用沉默逃避 |

**示例（审讯式）：**
```
"那天晚上你在哪里？"
"在家。"
"有人能证明吗？"
"...没有。"
"你是一个人？"
"是的。"
"整个晚上？"
```

**示例（反转权力）：**
```
"你应该告诉我真相。"
"应该？"她笑了。"你有什么资格跟我说'应该'？"
他一时语塞。
"等你把自己的事情交代清楚，再来问我的。"
```

## 对话场景类型

### 争吵场景

**特征**：短句、打断、重复强调、情绪升级

```
"你答应过的！"
"情况变了！"
"那是你的借口！"
"你根本不懂！"
"我当然不懂！你什么都不告诉我！"
```

### 告白场景

**特征**：迟疑、停顿、寻找词语、真诚或尴尬

```
"我...我想说...这些年，我一直在想...如果我们..."
她低下头，声音变小。
"如果我们什么？"
"如果我们早一点相遇。"
```

### 暧昧场景

**特征**：双关语、试探、暗示、话里有话

```
"你今天很漂亮。"
"只是今天？"
"嗯...今天特别漂亮。"
"那我明天该担心了？"
"明天...明天再看看。"
```

### 告别场景

**特征**：未尽之言、克制、表面平静下的暗涌

```
"那就这样吧。"
"嗯。"
"照顾好自己。"
"你也是。"
她转身走了两步，又停下来。"那盆茉莉记得浇水，别又忘了。"
"好。"
她没再回头。
```

## 对话常见问题

### 避免

1. **信息倾倒** - 角色互相说已知信息
   > 错误："正如你所知，我们的公司成立于1995年..."

2. **所有人说话一样** - 无法区分角色
   > 解决：给每个角色独特的说话方式（用词、句式、语气、口头禅）

3. **过度礼貌** - 真实对话更粗糙
   > 错误："我很抱歉打扰你，能否请你..."
   > 正确："喂。帮我个忙。"

4. **无意义的闲聊** - 除非有特殊目的

5. **说教** - 角色发表长篇哲学论述

## 对话练习自查

写完对话后检查：

- [ ] 每句对话是否有目的？
- [ ] 删除后情节是否受影响？
- [ ] 能否辨识说话人（不看标签）？
- [ ] 是否有潜台词？
- [ ] 节奏是否合适（快/慢）？
- [ ] 标签使用是否克制？
