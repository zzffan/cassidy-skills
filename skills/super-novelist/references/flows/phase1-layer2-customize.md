# 第一阶段·第二层：深度定制与创作规格（Q4-Q8）

> 每个问题都支持「🎲 随机生成」选项。选择后 AI 根据第一层已有信息智能生成合理答案，简短展示一句话供确认。
>
> 用户随时可以说"跳过"、"下一个"、"直接进Q8"来加速流程。

---

## Q4：世界观/背景设定

```
Question: 故事发生在什么样的世界？
Options:
- 现实世界（当代中国或其他现实背景）
- 架空历史（特定朝代或虚构朝代）
- 完全虚构世界（需自建规则体系，如魔法/修真体系）
- 未来/科幻世界（科技水平、社会结构特殊）
- ⭐自由描述（我已有详细设定）
- 🎲 随机生成（根据题材智能生成世界观）
```

**智能推荐逻辑**：
- 题材 = 奇幻/玄幻/科幻 → **强烈推荐（⚠️）**
- 题材 = 都市现实 → 可跳过（默认现实世界）
- Q1自由输入中已含世界观信息 → 标记"已覆盖"

**追问**（如果用户选择了需要展开的选项）：
```
Question: 这个世界有什么独特的规则或设定要素？（简单描述即可）
Options:
- [基于题材推荐的2-3个方向]
- 自由描述
- 暂时没想到，后面再定
```

**记录**：`世界观 = [用户选择/生成结果]`

## Q5：叙事视角与基调

```
Question A - 叙事视角：你希望用什么视角讲故事？
Options:
- 第三人称限制视角（跟随主角的所见所感）
- 第三人称全知视角（上帝视角，可切换角色）
- 第一人称（主角自述，代入感最强）
- 多视角切换（每章或每段切换不同角色视角）
- ⭐自由描述
- 🎲 随机生成
```

```
Question B - 整体基调：故事的氛围风格是？
Options:
- 紧张刺激（快节奏、高冲突、悬念驱动）
- 温情治愈（慢节奏、情感向、暖心）
- 轻松幽默（喜剧感、轻松愉快）
- 沉重深刻（探讨人性/社会、有文学性）
- ⭐自由描述
- 🎲 随机生成
```

> **偏好集成**：如果 `preferredPerspective` / `preferredTone` 有值，标记对应选项为⭐。

**记录**：`叙事视角 = [...]`，`基调 = [...]`

## Q6：核心主题/价值观

```
Question: 这部小说最想表达的核心主题是什么？（这决定了故事的灵魂）
Options:
- 成长与蜕变（主角的内在变化是核心）
- 正义与复仇（善恶对决、伸张正义）
- 爱与牺牲（感情线驱动，为所爱之人付出一切）
- 权力与欲望（野心、争夺、堕落与救赎）
- 生存与希望（绝境中的人性光辉）
- 自由与束缚（对抗命运/体制/偏见）
- ⭐自由描述（我有明确主题）
- 🎲 随机生成
```

**记录**：`核心主题 = [用户选择/生成结果]`

## Q7：读者定位与风格参考

```
Question A - 目标读者：主要写给谁看？
Options:
- 大众读者（番茄小说/网络文学受众，追求爽感和代入感）
- 青少年读者（偏青春、成长、校园题材）

- 没有 / 不确定
- 自由描述
- 🎲 随机生成
```

**记录**：`目标读者 = [...]`，`风格参考 = [...]`

## Q8：章节数量与特殊要求

```
Question A - 章节数量：你计划创作多少章？
Options:
- 10章（短篇，约3-5万字）
- 15章（中短篇，约4.5-7.5万字）
- 20章（中篇，约6-10万字）⭐
- 30章（中长篇，约9-15万字）
- 50章（长篇，约15-25万字）
- 自定义（输入具体数字）
```

> **偏好集成**：如果 `typicalChapterCount` 有值，标记对应选项为⭐并提示"上次你选了XX章"。

**记录**：`章节数 = [用户选择]`

**追问 — 特殊要求**（可选）：

```
Question: 有没有特殊要求？（均可跳过）
Options:
- 有必须包含的情节/场景（请描述）
- 有绝对不能出现的内容（请描述）
- 字数偏好（默认每章3000-5000字）
- 没有特殊要求，按标准来
- 自由描述多个要求
```

**记录**：`特殊要求 = [用户回答 / 无]`
