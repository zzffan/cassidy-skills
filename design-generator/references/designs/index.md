## Usage

To use any design system, reference it by name in your request:

```
"生成 Apple 风格的 landing page"
"创建一个 Spotify 风格的音乐播放器"
"用 Stripe 的设计风格做支付页面"
"做一个 Linear 风格的任务管理界面"
```

The skill will load the corresponding design specification and generate code that faithfully recreates that brand's design aesthetic.
